# Python Functions

This folder contains notes and example scripts demonstrating how to define and
use functions in Python.

Topics covered:
- Defining functions with `def`
- Parameters: positional, default, keyword-only, `*args` and `**kwargs`
- Return values and `None`
- Docstrings and type hints
- Lambda (anonymous) functions
- Recursion and tail-recursive patterns

Try the example scripts in this directory to see each concept in action:
- `basic_functions.py` — simple function definitions and calls
- `defaults_and_kwargs.py` — default args, `*args`, and `**kwargs`
- `recursion.py` — recursive examples (factorial, Fibonacci)
- `lambda_examples.py` — lambdas and higher-order functions
- `docstring_example.py` — docstrings and `help()` usage

Run an example with:

```bash
python3 basic_functions.py
```
